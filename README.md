# ReVAMS (Revenue and Attendance Management System)

![React](https://img.shields.io/badge/react-%2320232a.svg?style=for-the-badge&logo=react&logoColor=%2361DAFB) ![Next JS](https://img.shields.io/badge/Next-black?style=for-the-badge&logo=next.js&logoColor=white) ![TypeScript](https://img.shields.io/badge/typescript-%23007ACC.svg?style=for-the-badge&logo=typescript&logoColor=white) ![Supabase](https://img.shields.io/badge/Supabase-3ECF8E?style=for-the-badge&logo=supabase&logoColor=white) ![TailwindCSS](https://img.shields.io/badge/tailwindcss-%2338B2AC.svg?style=for-the-badge&logo=tailwind-css&logoColor=white) ![Status](https://img.shields.io/badge/Status-Discontinued-inactive?style=for-the-badge)

> **A modernized re-engineering of the Visayas State University Attendance Management System.**

ReVAMS is a web-based initiative designed to upgrade and streamline the attendance tracking process at VSU. Built with a focus on performance and scalability, it leverages Supabase for real-time data handling and Next.js for a responsive, server-side rendered user experience.

**Project Status:**
This project is currently **Discontinued** and is no longer being actively maintained. The repository serves as an archive of the development progress and architectural decisions made during the remake initiative.

---

## Key Features

* **Modern Interface:** A clean, responsive UI built with Tailwind CSS to replace legacy layouts.
* **Real-Time Database:** Utilizes Supabase for instant updates on attendance records and student logs.
* **Type-Safe Architecture:** Developed entirely in TypeScript to ensure code reliability and maintainability.
* **Efficient Querying:** Optimized data fetching strategies using Next.js.

---

## Tech Stack

* **Framework:** Next.js (React)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Backend/Database:** Supabase (PostgreSQL)
